Detailed Description

Qt table adapter exposing DICOM metadata rows to a QTableView.

Rows are loaded by calling Refresh() with a flat list of {field, value} pairs sourced from DicomController::GetMetaData(). The model notifies attached views via the standard QAbstractTableModel reset mechanism.
