Detailed Description

View widget for single-window multi-viewport DICOM display.

Implements ILayoutTarget: when ApplyLayout() is called, the renderer viewport rects on the single shared render window are updated.

If called before the VTK pipeline is ready (before first DICOM load), the layout is stored and applied automatically when ViewersReady fires.

Apply the given layout to this view.

Implementations must:

  1. Update the visual arrangement of their render surface(s).
  2. Request a render so changes appear immediately.

Called by ViewportLayoutManager whenever the active layout changes. Also called once after the pipeline is initialised so the initial layout is applied correctly.
