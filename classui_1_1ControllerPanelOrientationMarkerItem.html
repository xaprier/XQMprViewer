Detailed Description

Control panel item for anatomical orientation marker overlay.

Manages one or more OrientationMarkerOverlay instances simultaneously. All attached overlays are kept in sync with the panel settings.

Controls:

  • Enable / disable toggle (checkbox)
  • Long labels toggle: switches between L/R/A/P/S/I and full anatomical names
  • Font size (spin box, pt)
  • Label color (color picker button)
