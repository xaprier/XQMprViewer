Detailed Description

Control panel item for corner annotation overlay configuration.

Manages one or more CornerAnnotationOverlay instances simultaneously. All attached overlays are kept in sync with the panel settings.

Controls:

  • Enable / disable (checkbox)
  • Position (combo: same options as FPS overlay)
  • Show slice info (checkbox)
  • Show window/level (checkbox)
  • Show spacing (checkbox)
  • Show view name (checkbox)
  • Font size (spin box, pt)
  • Margin (spin box, px)
  • Text color (color picker button)
