Detailed Description

Manages up to three vtkResliceImageViewer instances for orthogonal slice display.

Supports two operating modes, detected automatically from the widget vector size:

  • 3 widgets → multi-window mode: each viewer owns a separate render window.
  • 1 widget → viewport mode: all viewers share one render window; interaction routing is handled externally by ViewportInteractorStyle.
