Detailed Description

Global orchestrator for runtime layout switching.

Owned by MainWindow. Drives layout changes across both views:

  • Calls ILayoutTarget::ApplyLayout() on every registered view.
  • Calls OverlayLayoutAdapter::Sync() on every registered adapter.
  • Keeps the current layout definition so new views can query it.

Decouples the selector widget (ViewportLayoutSelector / ViewportLayoutPanel) from both the view layer and the controller layer.

Overlay enable/disable events from the ControllerPanel must also pass through this manager so it can forward them to the correct adapters.

slotPlanes[i] = plane index (0=Axial,1=Coronal,2=Sagittal) for slot i. The geometry (viewport rects) stay the same; only which plane renders in each slot changes.
