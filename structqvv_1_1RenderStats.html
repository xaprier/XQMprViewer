Detailed Description

Per-viewport render performance statistics.

  • frameTimeMs — elapsed time between the last two frames (ms).
  • renderTimeMs — time spent inside vtkRenderWindow::Render() (ms).
  • frameCount — total number of frames rendered since creation.
