Detailed Description

View controller that displays three DICOM planes in a single render window.

Implements the same IViewController interface as MultiWindowController. Instead of three separate QVTKOpenGLNativeWidget instances, a single widget's render window is split into three equal horizontal regions via qvv::ViewportLayout, or any other runtime-selected layout via ApplyLayout().
