Detailed Description

Describes the viewport region and plane index for one pane in a layout.

planeIndex maps to SliceController::Orientation: 0=Axial, 1=Coronal, 2=Sagittal. -1 means the pane is unused / not rendered. slotIndex is the stable positional slot (0=first, 1=second, ...) used for per-slot plane assignment overrides.
