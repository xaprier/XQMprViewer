Detailed Description

Control panel item for FPS overlay configuration.

Manages one or more FPSOverlay instances simultaneously. All attached overlays are kept in sync with the panel settings.

Controls:

  • Enable / disable toggle (checkbox)
  • Corner position (combo: Top-Left, Top-Right, Top-Center, Bottom-Left, Bottom-Right, Bottom-Center)
  • Margin (spin box, pixels)
  • Text color (color picker button)
  • Font size (spin box, pt)
